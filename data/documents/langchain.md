# LangChain

LangChain is a framework for building applications powered by large language models.

It provides:

- Prompt Templates
- Chat Models
- LCEL
- Output Parsers
- Retrieval
- Agents
- LangGraph

Retrieval-Augmented Generation (RAG) combines document retrieval with LLMs to generate grounded responses.

Chroma is commonly used as a vector database in LangChain applications.
